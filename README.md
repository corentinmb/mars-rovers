# mars-rovers

To run the app, make a package:

`cd mars-rover/`

`mvn package` 

And you can use the app by passing an input file as parameter (for example the one I use for tests):

`cd target/`

`java -jar target/mars-rover.jar ../src/test/resources/input.txt`
